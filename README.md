PHPUnit Ansible role
=========

If you are a PHP developer and you want to install the latest version PHPUnit.

Requirements
------------
  - minimum `php 7.1` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phpunit

Check if it's installed:
------
    phpunit --version
